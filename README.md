# Bo Revenge

## Prerequisites:
- VSCode
- NodeJS
- Yarn
- [EsLint VSCode extension](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

## Local development:
- yarn install
- yarn dev
- open http://localhost:3000

## Run desktop version:
- yarn dev
- yarn dev:electron
